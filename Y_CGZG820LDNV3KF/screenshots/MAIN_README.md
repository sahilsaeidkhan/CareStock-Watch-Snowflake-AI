# CareStock Watch — Detailed Project Overview

CareStock Watch is a Snowflake-native inventory intelligence system built to help hospitals, NGOs, and public distribution systems predict stock-outs, reduce wastage, and prioritize procurement actions.

## Purpose
- Provide proactive early warnings for medicines and supplies
- Offer short-term demand forecasts and confidence bounds
- Enable data-driven procurement and action logging

## Key Features
- Real-time inventory health dashboard
- Days-to-stock-out and risk classification (RED / YELLOW / GREEN)
- Short-term AI-assisted demand forecasting (7–30 day horizons)
- Exportable reorder and prioritization lists
- Action logging and impact measurement
